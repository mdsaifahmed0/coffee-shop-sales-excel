# Coffee Shop Sales Analysis

## Tools Used
Microsoft Excel (Pivot Tables, Dashboard)

## Project Overview
Analyzed 149,116 coffee shop transaction records across 
3 store locations (Jan–Jun 2023) using Excel.

## Key Findings
- Revenue grew $81.7K → $166.5K (Jan–Jun, +103% growth)
- Coffee category led at $270K (38.6% of total revenue)
- Peak sales hours: 8–10 AM
- Hell's Kitchen: Top performing store ($236.5K)
- Barista Espresso: Best selling product ($91.4K)
